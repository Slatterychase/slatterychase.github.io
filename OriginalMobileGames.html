The Original Mobile Games - Second Avenue Learning

Game Engineer Intern - Unity, C#, Nintendo SDK

The Original Mobile Games is a physics based puzzle simulation game built in Unity based on historic handheld physical puzzles stored at the Strong Museum of Play in Rochester, New York. This project was created in partnership with the museum in order to preserve these games and allow them to be played. My work on this project primarily consisted of porting the game to the Nintendo Switch.

My work on this project included:

  • Adjusting gameplay mechanics to fit the Nintendo Switch control scheme, including adding features to handle joy con drift, updating gyro controls and adjusting for all controller states (Docked, Single Wireless, Dual Wireless), adjusted camera for different viewing modes (Docked, Standalone Screen, Handheld) to make the experience feel good for the player, and updated gameplay to work under Nintendo's guidelines for controller movement.
  • Optimized to meet industry standard benchmarks for performance and framerate.
  • Build new gameplay features, such as a timer and high score system, as well as adjusted existing ones in order to make the game stand out from the mobile version, such as adding stick controls to shake the board.
  • Created testing framework for efficient testing of win states across different puzzles both in Unity and on the development console.
